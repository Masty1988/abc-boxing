# Photos Disciplines

Ce dossier contient les photos statiques pour la page "Nos Disciplines".

## Photos nécessaires

### 1. Boxe Française
**Fichier** : `boxe-francaise.jpg`
- **Dimensions recommandées** : 1200 x 800 px (ratio 3:2)
- **Poids max** : 500 KB
- **Suggestions** :
  - Photo d'un tireur en action avec bottines visibles
  - Coup de pied technique (fouetté, chassé)
  - Ambiance élégante, technique
  - Bon éclairage, fond neutre ou salle

### 2. Kickboxing K-1
**Fichier** : `kickboxing-k1.jpg`
- **Dimensions recommandées** : 1200 x 800 px (ratio 3:2)
- **Poids max** : 500 KB
- **Suggestions** :
  - Photo d'action intense (coup de poing ou genou)
  - Pieds nus, shorts de combat
  - Énergie explosive, dynamique
  - Combat sur ring ou en entraînement

## En attendant les vraies photos

Vous pouvez utiliser temporairement :
- Une photo générique de la salle
- Une photo de combat existante
- Ou laisser le fond en dégradé (fonctionnel)

## Upload

Ces photos sont **statiques** (pas Cloudinary) :
1. Placer les fichiers directement dans ce dossier
2. Nommer exactement comme ci-dessus
3. Optimiser le poids avant upload (TinyPNG, Squoosh, etc.)
